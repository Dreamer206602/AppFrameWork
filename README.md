# AppFrameWork

- 好用的app框架，包含选项卡和侧滑
- 使用RadioGroup处理选项卡优化布局
- 使用ToolBar跟侧滑图表联动，效果更好
- 集合了viewerPager+fragment，drawerlayout+toolbar，optionmenu以及circleImageView处理头像等功能
- 详细介绍可以参考[我的csdn博客](http://blog.csdn.net/xsf50717/article/details/49799485)
